🎤 Sample Consultation Script

Read this script when recording to test the AI analysis:

Doctor: Good morning! What brings you in today? Patient: Hi doctor. I've been having severe headaches for the past week. They usually start in the morning and get worse throughout the day. Doctor: I see. Can you describe the pain? Is it throbbing or constant? Patient: It's more like a throbbing pain, mostly on the right side of my head. Sometimes I feel nauseous too. Doctor: Have you had headaches like this before? Patient: Yes, I had similar headaches about 6 months ago, but they went away after a few days. These ones are lasting longer. Doctor: Are you taking any medications currently? Patient: Just my blood pressure medication. Doctor: Okay. Let me check your blood pressure. Your blood pressure is slightly elevated at 145 over 90. Your pupils are reactive. No signs of neurological deficit. Doctor: Based on your symptoms and examination, this appears to be migraine headaches. I'm prescribing sumatriptan 50mg to take when you feel a headache coming on. Also, try to identify triggers - keep track of what you eat and your sleep patterns. Avoid caffeine and alcohol for now. Make sure you're drinking plenty of water. I want to see you back in two weeks to review your progress. Patient: Thank you, doctor.

🔍 What the AI Extracts

From the consultation audio, the AI intelligently extracts:

  • Patient Complaints: Main symptoms and chief complaints
  • History/Context: Medical history, symptom duration, contextual information
  • Observations: Doctor's examination findings and clinical notes
  • Advice/Recommendations: Treatment plans, prescriptions, lifestyle advice
  • Other Relevant Points: Additional important information
✅ Accuracy Guarantee: The AI extracts information ONLY from the audio content. It does NOT assume, infer, or fabricate any details. Missing information is clearly marked as "Not mentioned in audio".

⚙️ Setup Instructions (For First-Time Use)

1 Install Backend Dependencies

cd backend pip install -r requirements.txt

2 Set Google API Key (Optional but Recommended)

For AI-powered analysis, set your Google API key:

# Get your key from: https://makersuite.google.com/app/apikey # Windows PowerShell: $env:GOOGLE_API_KEY="your-api-key-here" # Windows CMD: set GOOGLE_API_KEY=your-api-key-here
⚠️ Note: Without an API key, the system will still work but will use basic transcription without intelligent AI analysis.

3 Install FFmpeg (For Audio Conversion)

Download from: ffmpeg.org

Add FFmpeg to your system PATH

4 Start the Servers

# Terminal 1 - Backend: cd backend python main.py # Terminal 2 - Frontend: cd frontend npm run dev

🔧 Troubleshooting

Issue: "Could not understand audio"
Solution: Ensure microphone is working, speak clearly, check audio quality
Issue: "Failed to access microphone"
Solution: Grant microphone permissions in your browser settings
Issue: "Audio conversion failed"
Solution: Install FFmpeg and ensure it's in your system PATH
Issue: Basic transcription without AI analysis
Solution: Set GOOGLE_API_KEY environment variable and restart the backend server
